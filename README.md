## 🚀 Project Overview

This project is created using **Microsoft Excel / Google Sheets** to demonstrate basic skills in **data entry, organization, and simple analysis**.

The main focus of this project is to understand how data is structured and managed in spreadsheets.

---

## 🎯 Objectives

* Learn how to enter and organize data properly
* Understand table structure in spreadsheets
* Perform basic calculations
* Build a foundation for advanced data analysis

---

## 🧠 Features of the Project

### 🔹 Data Handling

* Structured dataset with multiple entries
* Organized rows and columns
* Clean and readable format

### 🔹 Basic Calculations

* Simple arithmetic operations (like totals)
* Basic use of formulas

### 🔹 Data Organization

* Proper column naming
* Easy-to-understand layout
